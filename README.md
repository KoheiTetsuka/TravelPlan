## 機能説明
+ 各一覧画面
  + 登録した内容の一覧を閲覧、削除をすることができる
+ 各登録画面
  + 旅行プラン、スケジュール、持ち物の登録をすることができる
+ 各編集画面
  + 旅行プラン、スケジュール、持ち物の編集をすることができる

## 使用技術

+ Kotlin Coroutine
+ Room
